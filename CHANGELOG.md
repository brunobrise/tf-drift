# Changelog

All notable user-facing changes are tracked here.

## Unreleased

* Repository polish: README badges, screenshot, community metadata, and maintenance templates.

## v1.1.0 - 2026-06-20

* Added mouse scroll support and page up/down key aliases.
* Added scheduled release automation.
* Added selectable TUI styles.
* Added `-reconfigure` and `-migrate-state` init flags.
* Added local Terraform/OpenTofu status examples.
* Added interactive Terraform config selection.
* Added glob and brace expansion support in `-dir`.
* Fixed relative path display in the TUI.
* Improved TUI row highlighting and prompt behavior.
